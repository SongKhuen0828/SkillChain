# 组织创建流程

## 概述

当系统管理员创建新组织时，系统会：
1. 创建组织账户
2. 自动生成组织管理员账户（使用提供的联系邮箱）
3. 生成临时密码
4. **显示登录凭据供管理员手动分享**

## 重要说明

**邮件发送功能已移除**。系统不会自动发送欢迎邮件。创建组织后，系统管理员需要：

1. 复制显示的登录凭据（邮箱和密码）
2. 通过您自己的邮件服务（Gmail、Outlook 等）发送给组织管理员
3. 建议在邮件中提醒组织管理员首次登录后修改密码

## 创建组织步骤

1. 进入 **Admin Dashboard → Organizations**
2. 点击 **Create Organization**
3. 填写表单：
   - **Organization Name**（必填）
   - **Description**（可选）
   - **Website**（可选）
   - **Contact Email**（必填）- 将作为组织管理员的登录邮箱
   - **Contact Name**（可选）- 组织管理员的姓名
4. 点击 **Create Organization**
5. 创建成功后，会弹出对话框显示：
   - 组织管理员的登录邮箱
   - 临时密码
6. **复制这些凭据并手动发送邮件给组织管理员**

## 安全建议

- 临时密码是随机生成的，包含大小写字母、数字和特殊字符
- 建议提醒组织管理员首次登录后立即修改密码
- 不要在公开渠道分享这些凭据
- 建议通过加密邮件或安全通信渠道发送凭据

## 后续操作

组织管理员收到凭据后可以：
1. 使用提供的邮箱和密码登录
2. 首次登录后修改密码
3. 开始管理组织成员、创建课程等
